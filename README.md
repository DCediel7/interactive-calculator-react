# 🧮 Interactive Calculator React

A modern and interactive calculator built with **React** ⚛️
This project focuses on clean component architecture, reusable UI elements, input validation, and responsive design.

---

## ✨ Features

✅ Basic arithmetic operations
✅ Input validation
✅ Error handling
✅ Decimal support
✅ Clear and erase functions
✅ Responsive interface
✅ Reusable React components
✅ Organized project structure

---

## 🛠️ Technologies Used

* ⚛️ React
* ⚡ Vite
* 🟨 JavaScript (ES6+)
* 🎨 CSS3

## 🚀 Installation

Clone the repository:

```bash
git clone https://github.com/your-username/interactive-calculator-react.git
```

Go to the project folder:

```bash
cd interactive-calculator-react
```

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```
